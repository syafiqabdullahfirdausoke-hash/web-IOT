✨ Fitur Smart Logic

Sistem ini dapat membangun tampilan secara mandiri berdasarkan data yang dikirim dari Micro:bit. Gunakan format pengiriman Key:Nilai (contoh: Cahaya:100).

  • Kirim key "mode", "lampu", atau "otomatis" ➔ Otomatis menjadi widget Toggle Switch.
  • Kirim key "kipas" ➔ Otomatis menjadi widget Range Slider (Dimmer).
  • Key selain di atas ➔ Otomatis menjadi widget Grafik Line Chart.
